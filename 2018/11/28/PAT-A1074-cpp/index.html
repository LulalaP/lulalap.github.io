PAT A1074 Reversing Linked List(C++)

发表于 2018-11-28 | 分类于 PAT题解 | |
PAT甲级目录 | PAT乙级目录

题目描述

原题地址：A1074 Reversing Linked List
中文版：B1025 反转链表

解题思路

将单链表保存到数组中，用逆转函数对特定范围进行逆转，最后按格式输出

易错点

  • 给定单链表可能包含无用结点
  • 地址固定为五位
  • 最后一个结点的 next 输出 -1

也许陌生的知识点

  • reverse(S.begin() + first, S.begin() + last);
    • 对数组 [first, last) 范围内的元素逆转
    • 需要头文件：algorithm
